Porting to VRChat of Unity Localization package.

## Features

- バリアントモードへ対応

## Documentation

*Unity 2022.3+ が必要です*

### VCC を利用したインストール

1. 次のページを開き、「Add to VCC」を押します。  
  [HoshinoLabs VPM Repository](https://vpm.hoshinolabs.com/)
2. VCCの「Manage Project」を押す。
3. `Localization` の横の「+」ボタンを押す。
